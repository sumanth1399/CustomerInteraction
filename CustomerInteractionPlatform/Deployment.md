# Chat Service - Firebase Deployment Guide

## Prerequisites

1. Install Google Cloud SDK: https://cloud.google.com/sdk/docs/install
2. Authenticate: `gcloud auth login`
3. Set your Firebase project: `gcloud config set project YOUR_PROJECT_ID`
4. Enable billing for your Firebase project (required for Cloud Run)

## Deployment Steps

### Option 1: Using the deployment script (Recommended)

1. Make the script executable:
   chmod +x deploy.sh
